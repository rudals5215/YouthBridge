# YouthBridge 🚀

**청년 정책을 한 곳에서, 나에게 맞게 찾아주는 서비스**

YouthBridge는 복잡하게 흩어져 있는 청년 정책(지원금, 적금, 주거 지원 등)을
**사용자의 나이, 지역, 조건에 맞게 추천해주는 플랫폼**입니다.

정부와 지자체에서 제공하는 다양한 청년 정책들은 정보가 여러 사이트에 분산되어 있어
본인이 받을 수 있는 혜택을 놓치는 경우가 많습니다.

YouthBridge는 이러한 문제를 해결하기 위해
**정책 정보를 자동으로 수집하고, 사용자 조건에 맞는 정책을 추천하는 서비스**를 목표로 합니다.

---

# ✨ 주요 기능

### 1️⃣ 맞춤 정책 추천

사용자의 정보를 기반으로 받을 수 있는 정책을 자동으로 필터링합니다.

* 나이
* 거주 지역
* 성별
* 소득 조건 (확장 예정)

예시

> 부산 거주 / 만 28세 → 받을 수 있는 청년 정책 리스트 제공

---

### 2️⃣ 청년 정책 통합 조회

흩어져 있는 정책 정보를 한 곳에서 확인할 수 있습니다.

예시 정책

* 청년도약계좌
* 청년 월세 지원
* 청년 교통비 지원
* 청년 구직활동 지원금
* 지자체 청년 정책

---

### 3️⃣ 정책 상세 정보 제공

각 정책의 상세 정보를 확인할 수 있습니다.

* 지원 대상
* 지원 금액
* 신청 방법
* 신청 기간
* 공식 사이트 링크

---

### 4️⃣ 지역별 정책 필터링

지역별로 제공되는 청년 정책을 확인할 수 있습니다.

예시

* 부산 청년 정책
* 경남 청년 정책
* 울산 청년 정책

---

# 🏗️ 시스템 아키텍처

```
정부 정책 사이트
        │
        │ (크롤링)
        ▼
Crawler Server
        │
        ▼
Database
        │
        ▼
Backend API
        │
        ▼
Frontend (React)
```

### 데이터 수집 방식

사용자가 접속할 때마다 크롤링을 하면 속도가 느려지기 때문에
**정해진 시간마다 정책 정보를 미리 수집하여 DB에 저장합니다.**

예시

* 매일 새벽 3시 크롤링
* 정책 정보 업데이트
* 사용자 요청 시 DB에서 빠르게 조회
